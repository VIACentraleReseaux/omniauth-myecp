# Omniauth::Myecp

Enables authentification against MyECP-OAuth2 with Omniauth

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'omniauth-myecp'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install omniauth-myecp
